Bài 18: Thực hành tổng hợp thiết kế trang web

Tạo biểu mẫu đăng kí Hiệu ứng hover Thiết kế mới

Luyện tập 1: Tạo trang đăng kí

Lời giải:

  1. Tạo tệp dang_ki.html chứa biểu mẫu đăng kí câu lạc bộ.
  2. Ví dụ mã HTML cho biểu mẫu:
    3. <form action="/submit_form" method="POST">
    <label for="name">Họ và tên:</label>
    <input type="text" id="name" name="name" required><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required><br>
    <label for="club">Câu lạc bộ muốn tham gia:</label>
    <select id="club" name="club">
    <option value="tech">Công nghệ</option>
    <option value="art">Nghệ thuật</option>
    <option value="sports">Thể thao</option>
    </select><br>
    <button type="submit">Đăng kí</button>
    </form>

Luyện tập 2: Hiệu ứng hover

Lời giải:

/* Thay đổi màu chữ khi di chuột qua */
.nav a:hover {
color: #ff5722;
}

/* Thay đổi màu nền và viền khi di chuột qua các nút */
button:hover {
background-color: #ff5722;
border: 1px solid #e64a19;
}

Vận dụng: Thiết kế mới

Lời giải:

Tạo tệp style.css với thiết kế mới:

/* Định dạng banner */
.banner {
background: url("../assets/img/bg-masthead.jpg") no-repeat center center;
background-size: cover;
padding: 12rem 0;
text-align: center;
color: darkred;
}

/* Định dạng slogan */
.slogan {
background-color: rgb(248, 249, 250);
text-align: center;
padding: 7rem 3rem;
}

/* Định dạng khối */
.row {
display: flex;
flex-wrap: wrap;
max-width: 100%;
padding: 0 3rem;
}
.block_3 {
flex: 0 0 33.333%;
}
.block_3 h3 {
font-size: 24px;
font-weight: bold;
}
.text-content {
font-size: 16px;
color: #333;
}
